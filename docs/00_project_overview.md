# Project Overview

Solar Oracle Walkman is an experimental research and art-tech prototype.

It connects DIY DSSC fabrication, I–V curve measurement, material fingerprinting, oracle signatures, and blockchain records. The core research question is whether a physical energy/material event can become a verifiable identity signal.

This repository is currently centered on a V1 prototype: an IV voiceprint smart contract that validates and stores a compact measurement vector on-chain.

In later iterations, the broader project may include:
- richer measurement protocols,
- improved feature extraction,
- a public research registry,
- LLM-assisted explanation layers,
- and exhibition documentation.

This framing is intentionally precise and non-hype: it is a public research prototype, not a production certification platform.
